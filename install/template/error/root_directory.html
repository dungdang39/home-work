{{ app_name }} 설치 오류

프로젝트 경로에 .env 파일을 생성할 수 없습니다.


이는 파일 권한 문제 또는 디렉토리 권한 문제로 인해 발생할 수 있습니다. 서버 설정을 확인하고 아래 해결 방법을 참고하여 .env 파일을 생성하세요.

  1. Shell 명령어로 .env 파일 생성
    • 터미널을 열고 프로젝트 경로로 이동합니다.
    • 다음 명령어를 실행하여 설치 스크립트를 실행합니다: 
      $> php install/env.php
  2. 수동으로 .env 파일 생성
    • 터미널을 열고 프로젝트 경로로 이동합니다.
    • 다음 명령어를 실행하여 .env.example 파일을 복사해 .env 파일을 생성합니다: 
      cp .env.example .env
    • .env 파일을 열고 필요한 설정을 수동으로 입력합니다.
      • DB_CONNECTION: 사용할 데이터베이스 드라이버 (예: mysql)
      • DB_HOST: 데이터베이스 서버의 호스트명 (예: localhost)
      • DB_PORT: 데이터베이스 서버의 포트 번호 (예: 3306)
      • DB_DBNAME: 데이터베이스 이름
      • DB_USERNAME: 데이터베이스 사용자명
      • DB_PASSWORD: 데이터베이스 비밀번호
      • DB_PREFIX: 데이터베이스 테이블 접두사 (예: new_)
